# Horror Sound Files

This directory contains horror sound effects for the interactive horror novel game.

## Current Status
The sound system is implemented but requires actual audio files. Currently using placeholder paths.

## Required Sound Files

Please add the following MP3 files to this directory:

1. **ambient_creepy.mp3** - Background ambient horror sound (looping)
   - Suggested: Dark ambient, wind, distant sounds
   - Volume: Low (0.3)
   - Loop: Yes

2. **heartbeat.mp3** - Heartbeat sound effect
   - Suggested: Heavy heartbeat rhythm
   - Volume: Medium (0.5)
   - Duration: 3-5 seconds

3. **scream.mp3** - Scream/shock sound
   - Suggested: Horror scream or shock sound
   - Volume: Medium-Low (0.3-0.4)
   - Duration: 1-3 seconds

4. **door_creak.mp3** - Door opening sound
   - Suggested: Creaky door or metal door
   - Volume: Medium (0.6)
   - Duration: 2-3 seconds

5. **drip.mp3** - Water dripping
   - Suggested: Water drops echoing
   - Volume: Medium (0.6)
   - Duration: 2-5 seconds

6. **glitch.mp3** - Digital glitch effect
   - Suggested: Electronic glitch/distortion
   - Volume: Medium (0.6)
   - Duration: 1-2 seconds

7. **footsteps.mp3** - Footsteps approaching
   - Suggested: Heavy footsteps on concrete/wood
   - Volume: Medium (0.6)
   - Duration: 3-5 seconds

8. **breath.mp3** - Heavy breathing
   - Suggested: Slow, heavy breathing
   - Volume: Low (0.3)
   - Duration: 3-5 seconds

## Free Sound Resources

You can download free horror sounds from:
- Freesound.org
- Zapsplat.com (free tier)
- Pixabay (audio section)
- YouTube Audio Library
- BBC Sound Effects

## Note

The game will work without audio files (sounds will simply not play), but adding these files will greatly enhance the horror atmosphere.
